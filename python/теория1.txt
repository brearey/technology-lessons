Теория
- Чтобы создать файл используй команду touch НАЗВАНИЕФАЙЛА
- Питон файлы имеют расширение .py
- Чтобы проверить есть ли питон в компьютере выполни python3 --version (выйдет версия)
- Чтобы запустить питон файл выполни python3 НАЗВАНИЕФАЙЛА.py
- Чтобы вывести на питоне текст используй функцию print("ТВОЙ ТЕКСТ")
- Чтобы установить библиотеку ЧЕРЕПАШКА используй sudo apt-get install -y python3-tk
- Чтобы создать программу с черепашкой создай файл и напиши в нем, потом запусти:
    import turtle
    t = turtle.Turtle()
    t.forward(100)
    turtle.done()
