### 2. Теоретическая часть (10 мин)  
**Ключевые понятия (кратко, с примерами в терминале):**  

1. **Утилита `python3`**  
   - Проверка версии: `$ python3 --version`  
   - Запуск интерпретатора: `$ python3` (выход — `exit()` или `Ctrl+D`).  

2. **Утилита `pip3`**  
   - Менеджер пакетов Python: `$ pip3 --version`  
   - Установка пакета: `$ pip3 install имя_пакета`  

3. **Виртуальное окружение `venv`**  
   - Зачем нужно: изоляция зависимостей проекта.  
   - Создание: `$ python3 -m venv myenv`  
   - Активация: `$ source myenv/bin/activate` (в терминале появится `(myenv)`).  

4. **Файл `main.py`**  
   - Стандартный файл для кода Python.  
   - Создание через терминал: `$ touch main.py`  

5. **Программа «Hello, World!»**  
   - Код:  
     ```python
     print("Hello, World!")
     ```  
   - Запуск: `$ python3 main.py`  

6. **Внешние библиотеки**  
   - Пример: `turtle`
   - `sudo apt-get install -y python3-tk`

7. **Модуль `turtle`**  
   - Графическая «черепашка» для визуализации кода.  
   - Базовый синтаксис:  
     ```python
     import turtle
     t = turtle.Turtle()
     t.forward(100)
     turtle.done()
     ```

### 3. Практическая часть (25 мин)  
**Задание 1. Настройка окружения**  
1. Создайте папку для проекта:  
   ```bash
   $ mkdir python_project
   $ cd python_project
   ```  
2. Создайте виртуальное окружение:  
   ```bash
   $ python3 -m venv venv
   $ source venv/bin/activate
   ```  
3. Проверьте, что окружение активировано (в терминале есть `(venv)`).  

**Задание 2. Первая программа**  
1. Создайте файл `main.py`:  
   ```bash
   $ touch main.py
   ```  
2. Откройте файл в текстовом редакторе (например, `nano main.py`) и вставьте код:  
   ```python
   print("Привет, Python!")
   ```  
3. Запустите программу:  
   ```bash
   $ python3 main.py
   ```  

**Задание 3. Рисуем с черепашкой**  
1. Добавьте в `main.py` код для рисования квадрата:  
   ```python
   import turtle

   t = turtle.Turtle()
   for _ in range(4):
       t.forward(100)
       t.right(90)
   turtle.done()
   ```  
2. Запустите:  
   ```bash
   $ python3 main.py
   ```  
3. *Эксперимент:* измените длину сторон или углы, чтобы получить другой многоугольник.  

**Задание 4. Используем переменные и условия**  
1. Модифицируйте код, чтобы черепашка рисовала разные фигуры в зависимости от переменной:  
   ```python
   import turtle

   t = turtle.Turtle()
   shape = "triangle"  # попробуйте "square" или "circle"

   if shape == "triangle":
       for _ in range(3):
           t.forward(100)
           t.left(120)
   elif shape == "square":
       for _ in range(4):
           t.forward(100)
           t.right(90)

   turtle.done()
   ```  
2. Поменяйте значение `shape` и понаблюдайте за результатом.
