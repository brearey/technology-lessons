II Turtle
1. в своей папке создать файл myturtle.py
2. открыть любым редактором этот файл
3. добавить программу:
import turtle
t = turtle.Turtle()
t.width(5)
t.forward(100)
t.left(120)
t.forward(100)
t.left(120)
t.forward(100)
t.left(120)
turtle.done()
4. сохранить файл
5. запустить
